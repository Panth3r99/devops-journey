# DevOps Server Monitoring Script

## 📌 Overview
This is a simple Bash-based monitoring tool that checks whether servers are reachable using ping.

## 🚀 Features
- Monitor multiple servers
- Displays UP/DOWN status
- Logs results with timestamps
- Lightweight and easy to run

## 🛠️ Technologies Used
- Bash scripting
- Networking (ping, DNS concepts)
- Git & GitHub

## ▶️ Usage
Run the script:
```bash
./check_server.sh
